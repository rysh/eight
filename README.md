# 補足

- 科学とはHow to do Whatなので信仰の代わりにはなり得ない
- 信仰でしか救われない人が居るならその人達のために拠り所を作る
- 呪術や儀礼を廃し、人類が歴史の痛みから学んだ他者との関わり方の知恵を伝える
- 「祀る」とは「崇める（worship）」ことではない
- 「荒ぶる脅威（怨霊や自然）」を「鎮め」その力を「恵み」として受け取るための「共生技術」である
- 科学やAIもまた、扱いを間違えれば人を滅ぼす「祟るもの」になり得る
- 神を置くことで人は自分の位置を相対化できる
- 一切の歴史は書き換えずにそのまま受け止める
- 過去の人々を新しい価値感で裁かず敬う
- 既存の宗教的権威を否定せず調和をめざす(一神教のみ二律背反の原理を持つので取扱注意)
